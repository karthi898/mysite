Latest Article

OmniStudio Inspector Extension

The OmniStudio Inspector is a streamlined Chrome extension designed for Salesforce developers. It provides lighting-fast discovery of Data Mappers, Integration Procedures, OmniScripts, FlexCards and Flows within your org. Key highlights include:

  • Comprehensive Flow support - grouped by name with version sorting and colour-coded status indicators
  • Robust error handling and URL validation for seamless API calls
  • Detailed insights for OmniScripts including Language and combined Type/SubType columns
  • Per-org caching and settings to keep your data isolated across instances
  • Quick actions to open components directly in Lightning or Classic designers
